The Aquila admin area design and color scheme.

== Description ==

A chic black and red color scheme for WordPress Admin (v4.0 onwards). We use this plugin for the handover to our clients to clean up the admin area from unnecessary or potentially confusing items.

Many updates have been made to the admin area, including:

*   Black and red color scheme.
*   Dashboard metaboxes removed and cleaned up.
*   User "Profile" area cleaned up and simplified.
*   WordPress logos replaced with Aquila logos.
*   Current user role added as a body class.
*   Removed "How are you.." from the top-right corner.

== Installation ==

1. Upload aquila-admin-color-scheme to the /wp-content/plugins/ directory
2. Activate Aquila Admin Color Scheme through the 'Plugins' menu in WordPress
3. That's it! All settings are applied automatically and there's no need for any further configuration.

== Frequently Asked Questions ==

= Do I need to configure this plugin or change any settings? =

No. All settings are applied automatically once the plugin is activated and there are no settings that need to be configured.
